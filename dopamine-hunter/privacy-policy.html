What Data is Stored Locally

The following information is stored on your device using Chrome's local storage API:

  • Intervention Statistics: Total count, daily count, weekly count
  • User Preferences: Language selection, message style, monitored platforms
  • Milestone Records: Which achievement milestones you've reached (10, 50, 100, etc.)
  • Donation History: Record of when you clicked donation buttons (for statistics only)

Important: This data NEVER leaves your device and can be cleared at any time through Chrome's settings.

Network Requests

The extension makes limited network requests for the following purposes:

AI Message Generation

  • What: Requests to Cloudflare Workers AI API to generate personalized intervention messages
  • What's Sent: Only anonymized context (e.g., "user visiting YouTube" + message style preference)
  • What's NOT Sent: Your name, email, browsing history, or any personally identifiable information
  • Fallback: If the AI is unavailable, the extension uses pre-written messages stored locally

Third-Party Services

The extension includes links to the following optional third-party services:

Donation Platforms

  • Ko-fi: For international supporters (PayPal/credit cards)
  • WeChat Pay: For supporters in China
  • Alipay: For supporters in China

Important: These links are completely optional. Clicking them will take you to external websites that have their own privacy policies. We do NOT track whether you donate or how much.

Permissions Explained

The extension requests the following Chrome permissions. Here's why each one is needed:

Permission Why We Need It
storage Store your preferences and statistics locally on your device
activeTab Display intervention overlays when you visit monitored websites
scripting Inject content scripts to detect when you visit distracting platforms
host_permissions Monitor only the platforms YOU select (YouTube, TikTok, etc.)

Note: Even though we request broad host permissions, the extension only activates on platforms you explicitly enable in settings.
